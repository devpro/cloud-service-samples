### Stitch CLI

[stitch-cli Reference](https://docs.mongodb.com/stitch/import-export/stitch-cli-reference/)

- Install it with NPM

```bash
npm install -g mongodb-stitch-cli
```

- Check current version

```bash
stitch-cli --version
```
